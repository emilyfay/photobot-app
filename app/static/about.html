About PicPickr

PicPickr classifies photos as below average, above average, and significantly above average based only on the image pixel data. It was trained using 10,000 landscape photos from Flickr using the number of views on Flickr as an admittedly imperfect measure of the quality of the photo.

If a photo is photo is rated below average, PicPickr thinks that it would receive fewer than 20 views per year on Flickr. Photos rated significantly above average are predicted to receive more than 5,000 views per year. You can find more technical information about how PicPickr works in the slide deck.

Greg Novak (web site, LinkedIn) built PicPickr over a period of three weeks in September 2014 as part of the Insight Data Science program. The code for the web site and back end is available on GitHub.
